# /debug — Start Systematic Debugging

Triggers the systematic-debugging skill.

```
NO FIXES WITHOUT ROOT CAUSE FIRST
```

## Steps
1. Memory recall — search for similar past issues
2. Root cause — read errors, reproduce, trace data flow
3. Pattern analysis — find working example, compare
4. Hypothesis — one variable at a time
5. Fix + verify — test before and after

## 3-Attempt Rule
3 consecutive failures → STOP → reassess architecture
