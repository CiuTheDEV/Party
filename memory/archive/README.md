# Memory Archive

Archive area for compressed weekly `today.md` history and any future memory snapshots that should stay preserved without bloating the always-on working files.

## Rule

- `memory/today.md` stays as the rolling log for the current week.
- At the week boundary, compress the finished week into a smaller archive note here instead of keeping `today.md` as an ever-growing changelog.
- Do not delete meaningful history; compress and move it here.

## Suggested naming

- `today-YYYY-week-XX.md`
- `today-YYYY-MM-DD_to_YYYY-MM-DD.md`

Keep the archive readable, but optimize the main `memory/` files for fast session resume.
