# Nano Banan Game Art Prompts

Gotowy zestaw promptow do generowania spojnych grafik hero i kart gry dla Project Party.

## Zasady Wspolne

Te elementy powinny byc zachowane we wszystkich grafikach:

- jeden jezyk wizualny: premium dark neon
- tlo: gleboka czern, ciemny granat, subtelna mgla swietlna
- wspolny styl swiatla i kontrastu, ale paleta zalezy od gry
- styl: polished luminous illustration, premium digital poster
- bez tekstu w obrazie
- bez logo
- bez UI
- bez watermarkow
- bez fotorealizmu
- bez zagracenia

## Palety Gier

Kazda gra powinna miec wlasna dominujaca palete, ale nadal nalezec do tego samego produktu:

- `Kalambury`: fiolet + elektryczny blekit
- `Tajniacy`: blekit + czerwien
- `Mamy szpiega`: bursztyn + czerwien + brudny fiolet
- `5 Sekund`: pomarancz + zolc + magenta

## Prompt Bazowy Stylu

Uzywaj tego jako wspolnej bazy:

```text
Premium dark party-game key art for a modern browser game hub, cinematic neon illustration, deep black background, elegant luminous line art, subtle volumetric light, high contrast, premium and minimal, theatrical atmosphere, polished digital poster style, no text, no logo, no UI, no watermark, clean composition, visually striking but not cluttered, part of one consistent visual series for the same product, same lighting language, same premium neon illustration style, same dark cinematic atmosphere, with a game-specific color palette
```

## Negative Prompt

Jesli generator wspiera negative prompt, uzyj:

```text
text, letters, logo, watermark, interface, UI, button, photorealistic face, realistic hands, low contrast, washed out, flat composition, childish cartoon, 3d render, noisy background, clutter, weapons, gore
```
