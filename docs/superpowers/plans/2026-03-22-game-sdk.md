# Game SDK Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `@party/game-sdk` — a TypeScript-only package that defines the contract every game module must implement, then migrate hub to use it.

**Architecture:** Pure TypeScript types, no runtime code, no dependencies. Types are compiled to `.d.ts` files and consumed by hub and game packages. Hub's `games.ts` is migrated to import `GameConfig` from SDK and `config` from each game package.

**Tech Stack:** TypeScript 5, `tsc` compiler only
